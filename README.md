# Fashion Sales Analytics

Proyecto de análisis de productos y stock para un emprendimiento de moda.

El objetivo es aplicar conocimientos de Python y análisis de datos utilizando información de un catálogo real.

## Funcionalidades

- Lectura de productos desde un archivo CSV.
- Limpieza y validación básica de datos.
- Conteo de productos disponibles y sin stock.
- Cálculo del precio promedio.
- Cálculo del valor total del stock.
- Identificación del producto más caro y más barato.
- Análisis de productos por categoría.
- Generación de gráficos de precios y categorías.

## Tecnologías utilizadas

- Python
- pandas
- Matplotlib
- CSV
- Visual Studio Code
